# Guía de uso de cuaderno de Jupyter:

1. Colocar el archivo "Base_Violencia_INE_Unificada_LIMPIA.csv" en el folder "Cuaderno Jupyter para Analisis".
2. La primer celda del cuaderno instala los paquetes que falten, la segunda celda hace las importaciones. Si es necesario agregar más dependencias, agregarlas en estas primeras celdas (1, 2).
